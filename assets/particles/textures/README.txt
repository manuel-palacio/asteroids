particle.png — default libGDX particle dot texture

To populate this directory:
1. Copy particle.png from the libGDX Particle Editor distribution, or
2. Export it from the particle editor when you open a .p file, or
3. Create a simple 32x32 white circle PNG and name it particle.png

The .p files reference "particle.png" which must be in the same directory
(or configure ParticleEffectManager to point to the correct path).
